# exphon2026

2025년 실험음성학연구회 동계워크숍
2026년 1월 23일 (금요일)
한국외국어대학교

“강제음성정렬(Forced Alignment)과 음향 분석: wav에서 TextGrid를 거쳐 CSV까지”를 주제로 워크숍을 진행
- 실제 음성 데이터(.wav)와 전사 파일(.txt)을 이용해 강제음성정렬(Forced Alignment)을 수행하고, 정렬 결과를 Praat/Parselmouth로 처리하여 segment duration, F0, formant 등 음향 정보를 추출하고 CSV/엑셀 형식의 분석용 데이터셋으로 정리하는 전 과정을 함께 실습하는 것을 목표
- 실험음성학, 음운론, 음성공학 관련 대학원생 및 연구자분들이 강제음성정렬과 TextGrid 기반 음향 분석 파이프라인을 이해하고, 이후 석·박사 논문 및 연구 프로젝트에서 반복 가능한 자동화 분석 워크플로우를 설계하는 데 필요한 기본기를 갖추는 데 도움
